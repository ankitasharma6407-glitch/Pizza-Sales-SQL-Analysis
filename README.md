#  Pizza Sales SQL Analysis

# Project Overview

This project analyzes pizza sales data using SQL to extract meaningful business insights related to sales performance, customer preferences, and revenue trends.

The analysis was performed using MySQL and includes basic, intermediate, and advanced SQL queries involving joins, aggregations, CTEs, and window functions.



# Objectives

* Analyze pizza sales data using SQL.
* Calculate total orders and revenue.
* Identify customer ordering patterns.
* Determine the most popular pizzas and sizes.
* Analyze category-wise performance.
* Identify top revenue-generating products.



# Tech Stack

* MySQL
* SQL
* MySQL Workbench

* Orders

* SELECT Statements
* Aggregate Functions (SUM, COUNT, AVG)
* GROUP BY
* ORDER BY
* INNER JOIN
* Common Table Expressions (CTEs)
* Window Functions
* RANK()



# Business Questions Solved

### Basic Analysis

* Retrieve the total number of orders placed.
* Calculate total revenue generated from pizza sales.
* Identify the highest-priced pizza.
* Identify the most common pizza size ordered.
* List the top 5 most ordered pizza types.

# Intermediate Analysis

* Find total quantity of each pizza category ordered.
* Determine distribution of orders by hour.
* Analyze category-wise distribution of pizzas.
* Calculate average pizzas ordered per day.
* Determine top 3 pizza types based on revenue.

# Advanced Analysis

* Calculate percentage contribution of each pizza type to total revenue.
* Analyze cumulative revenue generated over time.
* Determine top 3 revenue-generating pizzas within each category.



# Key Findings

* Total Orders: **21,350**
* Total Revenue: **$817,860.05**
* Highest-Priced Pizza: **The Greek Pizza ($35.95)**
* Most Ordered Size: **Large (18,526 orders)**
* Most Ordered Pizza: **The Classic Deluxe Pizza (2,453 orders)**
* Highest Revenue Pizza: **The Thai Chicken Pizza ($43,434.25)**




## ✅ Conclusion

This project demonstrates how SQL can be used to transform raw sales data into actionable business insights. Through various SQL techniques, important trends related to sales performance, customer behavior, and product popularity were identified.
